# List-of-blocked-IPv6-addresses
Актуальный список заблокированных IPv6 адресов. Есть замечательный проект antifilter.download, откуда можно скачать список заблокированных адресов и подсетей, но, к сожалению, он генерирует/мониторит только IPv4. Хотелось бы иметь подобный актуальный список IPv6 адресов.
